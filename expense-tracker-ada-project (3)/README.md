

Expense management is an important aspect of personal financial planning. Recording daily income and expenses manually is time-consuming and may lead to errors. To overcome these problems, an Expense Tracker application has been developed using HTML, CSS, and JavaScript. The application enables users to maintain their financial records efficiently by allowing them to add, update, delete, search, and sort expense details.

The project demonstrates the practical implementation of Analysis of Design and Algorithms (ADA) concepts by integrating Linear Search, Binary Search, and Bubble Sort algorithms. Linear Search is used to search records sequentially, Binary Search performs faster searching on sorted records, and Bubble Sort arranges expense records in ascending order. The application also includes algorithm visualizations that help users understand the working of these algorithms and compare their execution time and complexity.

All expense data is stored using LocalStorage, eliminating the need for an external database. The project provides an easy-to-use interface with responsive design, making it suitable for both learning algorithm concepts and managing personal expenses. This application serves as an educational tool for understanding algorithm efficiency while providing a practical solution for expense management.

Keywords: Expense Tracker, Analysis of Design and Algorithms, Linear Search, Binary Search, Bubble Sort, HTML, CSS, JavaScript, LocalStorage.
