# ExpenceTracker-ADA
An Expense Tracker is a web-based application designed to help users organize and manage their daily financial transactions. It provides an efficient way to record income and expenses, monitor spending habits, and maintain financial records digitally. Compared to manual methods, the application reduces errors, saves time, and improves the accuracy of financial management.

The application is developed using HTML, CSS, and JavaScript, with LocalStorage used to store transaction data locally in the browser. It allows users to add new expense records, edit existing records, delete unwanted transactions, search for specific records, sort expenses, and view summaries. The user-friendly interface makes the application simple and easy to operate.

To demonstrate the practical application of Analysis of Design and Algorithms (ADA), the project implements three important algorithms:

Linear Search: Searches expense records one by one until the required record is found. It is simple to implement and works well for small datasets.
Binary Search: Searches sorted records by repeatedly dividing the search space into two halves, providing faster searching than Linear Search.
Bubble Sort: Arranges expense records in ascending order by repeatedly comparing and swapping adjacent elements until the list becomes sorted.

The project also includes interactive visualizations of these algorithms, allowing users to observe each step of their execution and compare their performance based on the number of comparisons and execution time.

Objectives
Develop a user-friendly Expense Tracker application.
Store and manage expense records efficiently.
Implement Linear Search for sequential searching.
Implement Binary Search for efficient searching in sorted data.
Implement Bubble Sort for sorting expense records.
Compare the performance of searching algorithms.
Demonstrate the practical application of ADA concepts.
Provide an educational and interactive learning platform.
Conclusion

The Expense Tracker successfully combines personal finance management with the concepts of Analysis of Design and Algorithms. By implementing Linear Search, Binary Search, and Bubble Sort, the application not only helps users manage expenses but also demonstrates how different algorithms perform in real-world scenarios. The project enhances understanding of algorithm efficiency while providing a practical, responsive, and easy-to-use expense management system suitable for academic learning and project demonstration.
